# EC2
Elastic Compute Cloud = Infrastructure as a Service
it mainly consists in th capability of:
  - Renting vitrual machines (EC2)
  - storing data on virtual drives (EBS)
  - Distributing load across machines (ELB)
  - Scaling the srvices using an auto-scaling group (ASG) 

## EC2 Sizing & configuration options
  - OS: Linux, Windows, Mac
  - CPU
  - RAM
  - Storage space: 
    - Network-attached (EBS & EFS)
    - Hardware (EC2 Instance Store)
  - Network card: speed of the card, Public IP address
  - Firewall rules: security group
  - Booststrap script (Configure at first launch): EC2 User Data

## EC2 User Data
- It is possible to bootstrap our instance using an EC2 User data script.
- Bootstrapping means launching commands when a machine starts.
- That script is only run once at the instance first start
- EC2 user data is used to automate boot tasks such as:
  - Installing updates
  - Installing software
  - Downloading common fiels from the internet
  - Anything you can think of
- The EC2 User Data Script runs with the root user

- When is using?

| 포인트        | 기억해야 할 핵심                                      |
| ---------- | ---------------------------------------------- |
| 언제 실행?     | **EC2 최초 부팅에서만 자동 실행**                         |
| 무엇을 위해 사용? | **서버 부팅 시 자동 설정 / 설치 / 스크립트 실행**               |
| 누가 실행?     | **Root 권한으로 실행**                               |
| 어디서 입력?    | **EC2 Launch할 때 Advanced details > User data** |
| 재부팅 시 실행?  | **기본적으로 NO (설정하면 가능하지만 기본값은 아님)**              |
| 주요 사용 목적   | 설치 자동화 / 구성 자동화 / 부팅 작업 자동화                    |
| OS 여부      | Linux Bash, Windows PowerShell 가능              |
